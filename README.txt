Module: MadisonLogic

Description
===========
Content Consumption Monitoring (CCM) is a technology that monitors B2B content consumption activities.

Installation
============
Put the modules into your sites/module folder
Enable the module and set user permissions

Configuration
=============
Visit /admin/config/system/madisonlogic
or via menus: Configuration > System > MadisonLogic

Set the required fields.
For the Email fields you can use any combination of #idemail, .classemail, *nameemail" separated by comma.
You can implement the hook_madisonlogic_email() to provide the parameter for the email address.
